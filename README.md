# Domour Copilot

[简体中文](./README-zh.md)

---

Domour Copilot is your personalized AI-powered coding assistant, designed to enhance development efficiency and make programming more intuitive. Integrated with state-of-the-art AI models, it deeply understands your code context to provide smart suggestions, automate repetitive tasks, and accelerate your workflow.

### Key Features

*   **Intelligent Code Completion**: Precise code suggestions based on your current context.
*   **Multi-language Support**: Supports major programming languages and frameworks.
*   **Cross-platform**: Seamless installation for Windows, macOS, and Linux.
*   **Elegant UI**: Clean and sophisticated interface based on Fluent Design.

### Quick Start

#### Terminal Installation (Recommended)

**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://domour.com/install.ps1 | iex"
```

**macOS / Linux (curl):**
```bash
curl -fsSL https://domour.com/install.sh | sh
```

#### Manual Download

Visit our [official website](https://domour.com) to download the specific installer for your platform (`.exe`, `.dmg`, `.deb`/`.rpm`).

### Development

The landing page is built with Vite, React, TypeScript, and Fluent UI v9.

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
