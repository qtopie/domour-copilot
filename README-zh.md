# Domour Copilot

[English](./README.md)

---

Domour Copilot 是一款为您量身定制的 AI 编程助手，旨在提升开发效率，让编程变得更加轻松。它集成了最先进的 AI 模型，能够深入理解您的代码上下文，提供智能建议、自动化重复任务并加速您的开发流程。

### 核心特性

*   **智能代码补全**：基于上下文提供精准的代码建议。
*   **多语言支持**：支持主流编程语言和开发框架。
*   **跨平台安装**：支持 Windows, macOS 和 Linux。
*   **极致体验**：简洁、优雅的 Fluent Design 界面。

### 快速安装

#### 命令行安装 (推荐)

**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://domour.com/install.ps1 | iex"
```

**macOS / Linux (curl):**
```bash
curl -fsSL https://domour.com/install.sh | sh
```

#### 手动下载

您可以前往我们的 [官方网站](https://domour.com) 下载对应平台的安装包 (`.exe`, `.dmg`, `.deb`/`.rpm`)。

### 开发与预览

本项目前端页面基于 Vite + React + TypeScript + Fluent UI v9 构建。

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
